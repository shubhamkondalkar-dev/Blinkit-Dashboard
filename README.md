# Blinkit-Dashboard
Excited to share my latest data analytics project where I analysed @Blinkit grocery data using Microsoft Excel and Power BI. 
 
 In this project, I worked on understanding sales performance, customer preferences, and outlet-wise trends through detailed data cleaning, and visualization. 
 
 The Power BI dashboard highlights key KPIs such as Total Sales, Average Sales, Number of Items, and Average Rating, along with interactive charts for fat content, item type, outlet size, and location-based analysis. 
 
 🔹 What makes this dashboard dynamic:
 ✔ Interactive slicers for outlet location and item type
 ✔ Clicking on one visual updates all related charts
 ✔ Drill-down analysis for better business understanding
 ✔ Real-time comparison of multiple KPIs
 
 🔹 Project Workflow:
 ✅ Data Cleaning & Preprocessing (Power Query)
 ✅ Data Modelling
 ✅ DAX Calculation 
 ✅ Sales Analysis & Visualization
 ✅ Power BI Dashboard Development
 ✅ Insight Generration

 
 🔹 Business Objective:
 To analyze BlinkIT’s sales performance, customer satisfaction, and inventory distribution to identify key insights and optimization opportunities using KPIs and visualizations. 
 
 📊 Key Insights:
 • Fruits & Vegetables and Snack Foods are top-selling categories
 • Low-Fat products contribute major revenue
 • Tier-3 locations show highest sales performance
 • Supermarket Type1 outlets dominate overall revenue
